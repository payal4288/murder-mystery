# 🔍 Murder Mystery Detective — AI-Powered Investigation Game

An AI-powered murder mystery detective game built with **Python**, **Streamlit**, and **Google Gemini AI**.

Every case is uniquely generated by Gemini — no two games are alike!

## 🎮 How to Play

1. Click **Start New Case** in the sidebar
2. Read the **Crime Scene Briefing** carefully
3. **Interrogate suspects** to hear their alibis
4. **Examine evidence** in the Evidence Locker
5. **Analyze contradictions** between clues and alibis
6. When you're ready, **make your accusation**
7. See the results and your detective rank!

## 📊 Scoring

| Action | Points |
|--------|--------|
| Examine a clue | +5 |
| Interrogate a suspect | +5 |
| Discover a contradiction | +20 |
| Correct accusation | +100 |
| Wrong accusation | -50 |

## 🚀 Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Get a Gemini API Key

1. Go to [Google AI Studio](https://aistudio.google.com/apikey)
2. Create a new API key
3. Open the `.env` file and replace `your_gemini_api_key_here` with your key:

```
GEMINI_API_KEY=your_actual_key_here
```

### 3. Run the game

```bash
streamlit run app.py
```

The game will open in your browser at `http://localhost:8501`.
